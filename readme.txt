**DECKHUD** - Custom TF2 HUD built for ease-of-play on Steam Deck

DeckHUD is a custom HUD designed to be close to the stock HUD, while improving on many quality-of-life features to make playing TF2 on Steam Deck more enjoyable.

- Many HUD elements have been enlarged
- Fonts have been made significantly clearer and sharper
- Critical HUD elements have been moved closer to the centre of the screen
- Visual de-clutter

**NOTE: DECKHUD IS CURRENTLY AN ALPHA RELEASE, A LOT OF CONTENT IS MISSING AND/OR PLACEHOLDER**
